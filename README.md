# Bidding-System


[![Travis](https://travis-ci.org/Omi10/BiddingSystem.svg?branch=master)](https://github.com/Omi10/BiddingSystem)

This project for Online Bidding System where users can upload their items for bidding

# Technologies Used

- Spring Boot
- Hibernate ORM
- Maven
- MySQL 5.6
- HSQL DB for Testing
- Javascript, HTML, CSS
